th:each

th:each 를 통해 태그들을 반복할 수 있다.

이름 나이 성별 주소

th:each 에 stat 을 추가해서 반복상태를 확인할 수 있다.

th:each 에 stat 을 추가하지 않으면, 변수면 + stat 으로 확인할 수 있다. stat.index -> memberStat.index

index: 0부터 시작하는 인덱스 값
count: 1부터 시작하는 값
current : 현재 객체의 정보
even : 짝수 번째 데이터 여부
odd : 홀수 번째 데이터 여부
first : 첫 번째 데이터 여부
last : 마지막 데이터 여부

이름 나이 성별 주소 INDEX COUNT SIZE CURRENT EVEN ODD FIRST LAST
